# Second Brain — Data Engineering Knowledge Hub

**Live Site:** [https://redgerd.github.io/Second-Brain/](https://redgerd.github.io/Second-Brain/)

**Second Brain** is my personal digital garden built with Quartz, where I document, connect, and grow my knowledge in **Data Engineering**.

Instead of keeping isolated notes, this project focuses on building a **connected knowledge system** — where ideas evolve, link together, and become long-term assets.

## Inspiration

This project is inspired by the public second brain by Sspaeti:

🔗 [https://github.com/sspaeti/second-brain-public](https://github.com/sspaeti/second-brain-public)

It demonstrates how a **personal knowledge system** can evolve into a public digital garden.
This repository follows a similar philosophy:

> Learn → Write → Connect → Share

## Purpose

This project helps me:

- Organize my learning in Data Engineering
- Build a long-term, searchable knowledge base
- Document concepts, tools, and real-world patterns
- Create a public portfolio of structured knowledge

## About Quartz

This project is powered by Quartz 4 — a set of tools that helps you publish your digital garden and notes as a website for free.

🔗 Documentation: [https://quartz.jzhao.xyz/](https://quartz.jzhao.xyz/)

## ⚙️ Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run locally

```bash
npx quartz build --serve
```

Starts a local development server to preview your digital garden.

### 3. Sync changes

```bash
npx quartz sync --no-pull
```

Pushes your updates without pulling remote changes (recommended for this setup).
